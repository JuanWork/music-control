
# 🎵 Music-Control (Universal Flow)

> **"Humanizando o desenvolvimento. Sinfonia e paz no código."**

Controle de áudio nativo para Linux (Rhythmbox) construído com **Node.js**, **AOP (Aspect-Oriented Programming)** e rodando em uma infraestrutura híbrida de **SSD + HD de 1TB**.

## 🚀 Arquitetura da "Casa"
- **Core:** Node.js (Proxy & Promises)
- **Logs de Humor:** Sistema de rastreamento de `Mood` (Calma vs Adrenalina)
- **Storage:** Persistência de logs e dados no HD de 1TB (`/mnt/armazenamento`)
- **UI:** Terminal interativo com cores ANSI (Azul e Verde)

## 🛠️ Como rodar a "Trinha"
Para validar o sistema e sentir a sinfonia:
